# Mini Web Service (Test Task) от Ялаев Р.Р.

##Требования:
- .NET 8.0
- Visual Studio 2022

##Как запустить:
Открыть VS и нажать F5
открыть в браузере http://localhost:5000/swagger 

##Решение состоит из двух проектов:
1)AdPlatforms.Core содержащий основную логику
2)AdPlatforms.Api содержащий логику запросов и хранения данных

##Данные хранятся в словаре с ключем являющимся локацией, и значением являющимся множеством платформ на этой локации.

##в Core создан класс LocationUtils с методами:
1)NormalizePath() нормализует строку с локациями
2)BuildIndexFromText() собирает словарь из текста
3)GetAllParentPaths() собирает из одного пути массив с его префиксами
4)FindPlatforms() находит все платформы на локации

##в Api хранилище через singleton, хранящий словарь и позволяющий атомарную замену снапшота. Так-же логика 2ух запросов load и find.
